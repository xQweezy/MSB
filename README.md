# 📰 Telegram‑бот для суммаризации статей

Бот извлекает текст из новостной статьи и с помощью **локальной нейросети** возвращает краткое содержание на русском языке. Поддерживает выбор длины пересказа, историю запросов и кэширование.

## ✨ Возможности

- ✅ Парсинг ссылок (РИА Новости, ТАСС, РБК, Хабр и др.)
- 🧠 Локальная модель `cointegrated/rut5-base-absum` (русский язык)
- 📏 Три режима длины пересказа: **кратко**, **средне**, **подробно**
- 📜 История последних 5 ссылок
- ⚡ Кэширование – повторные запросы обрабатываются мгновенно
- 🖱 Постоянная reply‑клавиатура + инлайн‑кнопки для выбора длины
- 🛡 Работа через SOCKS5‑прокси (опционально)

## 🚀 Установка и запуск

1. **Клонируйте репозиторий**  
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/telegram_summarizer_bot.git
   cd telegram_summarizer_bot
2. Создайте виртуальное окружение
python -m venv venv
source venv/bin/activate        # Linux/macOS
venv\Scripts\activate            # Windows
3. Установите зависимости
pip install -r requirements.txt
4. Настройте переменные окружения
Создайте файл .env и добавьте в него:
BOT_TOKEN=ваш_токен_от_BotFather
BOT_PROXY=socks5://логин:пароль@ip:порт   # если нужен прокси, иначе удалите строку
5. Запустите бота
python bot.py
При первом запуске модель автоматически скачается (≈1.4 ГБ). Время загрузки зависит от интернета, последующие запуски будут быстрее.

🧠 Как это работает
Пользователь отправляет ссылку.

trafilatura извлекает чистый текст.

Модель rut5-base-absum генерирует краткое содержание.

Результат сохраняется в SQLite‑кэш.

Под ответом появляется кнопка «✏️ Изменить длину», которая позволяет выбрать новый объём пересказа.
